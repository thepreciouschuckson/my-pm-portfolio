Brassmoney: Target Savings Feature Improvement

Focus: User research, UX optimization, and metrics.

The Challenge: A 20% setup abandonment trend linked to inflexible deduction timing (fixed at 8:30 AM), creating friction for users whose cash flow did not align.

The Execution: Conducted targeted user research with 20+ fintech users. Proposed a flexible UI solution with backend logic requirements for variable scheduling, aiming to increase setup completion and retention rates.
